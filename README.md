## Project Description

This project is a front-end restaurant website that showcases a home page, menu, and contact page. The goal is to provide a clean and modern web presence for a restaurant. It’s a static site, suitable for portfolio use or for small restaurants needing a basic web presence.

## Features

- Responsive layout (desktop, tablet, mobile)  
- Navigation bar between pages  
- Menu section with different categories (e.g., appetizers, mains, desserts)  
- Contact section (could include a form or contact details)  
- Clean design and simple interactions

## Usage

Once the project is open in your browser (or running via a local server), you can:

- Navigate between pages via the menu  
- View the restaurant’s menu section  
- Use the contact page to display contact details  

Feel free to fork the repository and customise it with your own design, images, or content.
